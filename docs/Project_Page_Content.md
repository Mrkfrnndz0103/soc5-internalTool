# Project_Page_Content

This file defines the content expectations for each page route.

## Dashboard (`/dashboard`)

- KPI summary cards (total dispatches, active routes, team members, average processing time).
- Recent activity feed.
- Quick action tiles.

## Outbound

### Dispatch Monitoring (`/outbound/dispatch-monitoring`)
- Status summary cards.
- Live dispatch feed with filters.

### Dispatch Report (`/outbound/dispatch-report`)
- Editable dispatch table.
- Validation and autosave hints.
- Submit confirmation.

### Prealert (`/outbound/prealert`)
- Filterable table with export.

### Per Bay Allocation (`/outbound/bay-allocation`)
- Allocation grid with quick edit actions.

## Data Team

### Prealert (`/data-team/prealert`)
- Same base table with data-team controls.

### SOCPacked Update (`/data-team/socpacked-update`)
- Import and reconcile workflow.

### File Upload (`/data-team/file-upload`)
- Upload zone with validation.

### Validation - Stuckup (`/data-team/validation/stuckup`)
- Table with exceptions, action controls.

### Validation - Shortlanded (`/data-team/validation/shortlanded`)
- Table with exceptions, action controls.
