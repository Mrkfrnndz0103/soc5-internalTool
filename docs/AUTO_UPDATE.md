# AUTO_Update

The documentation system supports automated updates during development.

## What it does

- Syncs documentation stubs from code changes.
- Updates documentation metadata when running the scripts.

## Scripts

- `npm run docs:update` updates documentation once.
- `npm run docs:watch` watches for changes and updates docs.

## Recommended Usage

- Run `docs:watch` during active development.
- Use `docs:update` before release.
